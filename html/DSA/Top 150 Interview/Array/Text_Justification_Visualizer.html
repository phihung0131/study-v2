Text Justification

LeetCode #68 | Difficulty: Hard | String, Simulation

📑 Problem Statement

Given an array of strings words and a width maxWidth, format the text such that each line has exactly maxWidth characters and is fully (left and right) justified.

Rules for Standard Lines

  • Pack as many words as possible.
  • Distribute spaces as evenly as possible.
  • Extra spaces go to the left slots.

Rules for Last Line

  • Left-justified only.
  • No extra space between words.
  • Pad with spaces at the very end.
Approach: Greedy Line Construction

Intuition: "The Packing Logic"

We process words greedily. For each line, keep adding words until the sum of word lengths + minimum spaces (1 between each) exceeds maxWidth. Once a line is "full", apply the spacing logic based on whether it is a normal line, a single-word line, or the last line.

Java Implementation

public List<String> fullJustify(String[] words, int maxWidth) {
    List<String> result = new ArrayList<>();
    int left = 0;
    
    while (left < words.length) {
        int right = findRightBoundary(left, words, maxWidth);
        result.add(justifyLine(left, right, words, maxWidth));
        left = right + 1;
    }
    
    return result;
}

private int findRightBoundary(int left, String[] words, int maxWidth) {
    int right = left;
    int currentLineWidth = words[left].length();
    
    while (right + 1 < words.length && (currentLineWidth + 1 + words[right + 1].length()) <= maxWidth) {
        right++;
        currentLineWidth += 1 + words[right].length();
    }
    return right;
}

private String justifyLine(int left, int right, String[] words, int maxWidth) {
    // Case 1: Single word in line or Last line
    if (right == left || right == words.length - 1) {
        return leftJustify(left, right, words, maxWidth);
    }
    
    // Case 2: Full justification with even spacing
    int numWords = right - left + 1;
    int totalSpaces = maxWidth - totalWordsLength(left, right, words);
    int spacesBetween = totalSpaces / (numWords - 1);
    int extraSpaces = totalSpaces % (numWords - 1);
    
    StringBuilder sb = new StringBuilder();
    for (int i = left; i <= right; i++) {
        sb.append(words[i]);
        if (i < right) {
            int count = spacesBetween + (i - left < extraSpaces ? 1 : 0);
            sb.append(" ".repeat(count));
        }
    }
    return sb.toString();
}

Complexity Analysis

  • ⏱️ Time: O(N). We traverse words once and characters once.
  • 📦 Space: O(maxWidth) per line for string construction.

The Spacing Trick

To distribute S spaces among G gaps:
Base spaces = S / G.
Leftover spaces = S % G.
First S % G gaps get (S / G) + 1 spaces. The rest get S / G.

⚡ Recall Cheat Sheet

  • • Pattern: Two-Pointer + StringBuilder.
  • • Step 1: while loop to find how many words fit.
  • • Step 2: Calculate gaps = count - 1.
  • • Step 3: If gaps == 0 or lastLine, pad right.
  • • Step 4: Otherwise, use / and % to distribute spaces.

🧩 Common Pitfalls

  • Forgetting to handle numWords == 1 separately (causes Division by Zero).
  • Not including the mandatory 1 space between words when checking maxWidth.
  • Incorrectly justifying the very last line (it must be left-aligned).
