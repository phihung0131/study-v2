Reverse Words in a String

LeetCode #151 | Difficulty: Medium | String Manipulation

📝 Problem Statement

Given an input string s, reverse the order of the words. A word is defined as a sequence of non-space characters. The words in s will be separated by at least one space.

Constraints to Handle:

  • Input may have leading or trailing spaces.
  • There may be multiple spaces between words.
  • The output should have only a single space separating the words and no extra spaces.

Example: " hello world " → "world hello"

Approach 1: Built-in Methods (Standard)

Intuition: "The Library Way"

Modern programming languages have powerful string manipulation tools. We can trim the edges, split by any amount of whitespace using regex, reverse the resulting array, and join them back together.

Java Implementation

public String reverseWords(String s) {
    // 1. Remove leading/trailing spaces and split by one or more spaces
    String[] words = s.trim().split("\\s+");
    
    // 2. Reverse the array of words
    Collections.reverse(Arrays.asList(words));
    
    // 3. Join them with a single space
    return String.join(" ", words);
}

Complexity

Time: O(n). Split and join both take linear time.

Space: O(n). Storing the array of split words takes extra space.

Trade-off

Very clean and readable. However, splitting creates many intermediate String objects, which might be inefficient for extremely large inputs or memory-constrained environments.

Approach 2: Two-Pointer Reverse Scan

Intuition: "Collect from the End"

Since the last word of the input should be the first word of the output, we can scan the string backwards. We find the end of a word, then search for its beginning, and append that segment to our result.

Java Implementation

public String reverseWords(String s) {
    StringBuilder resultBuilder = new StringBuilder();
    int rightPointer = s.length() - 1;

    while (rightPointer >= 0) {
        // 1. Skip trailing spaces
        while (rightPointer >= 0 && s.charAt(rightPointer) == ' ') {
            rightPointer--;
        }
        if (rightPointer < 0) break;

        // 2. Find the start of the word
        int leftPointer = rightPointer;
        while (leftPointer >= 0 && s.charAt(leftPointer) != ' ') {
            leftPointer--;
        }

        // 3. Append word and a space if not the first word added
        if (resultBuilder.length() > 0) resultBuilder.append(" ");
        resultBuilder.append(s.substring(leftPointer + 1, rightPointer + 1));

        // 4. Move rightPointer to check for next word
        rightPointer = leftPointer;
    }
    return resultBuilder.toString();
}

Complexity

Time: O(n). Each character is visited at most twice.

Space: O(n). StringBuilder stores the result.

Verdict

Interview Gold. This avoids creating an intermediate array and demonstrates mastery of pointer-based string traversal.

Approach 3: Double Reversal (Clever Trick)

Intuition: "The Mirror Logic"

This is a three-step technique often used to avoid extra space in languages with mutable strings (like C++). In Java, we use a char array:
1. Reverse the whole string: "the sky blue" → "eulb yks eht"
2. Reverse each word: "eulb" → "blue", "yks" → "sky"...
3. Clean up spaces: Remove duplicates and padding.

Step 1: Reverse the entire string character by character.

Java Implementation

public String reverseWords(String s) {
    char[] charArray = s.toCharArray();
    int n = charArray.length;
    
    // 1. Reverse the entire array
    reverse(charArray, 0, n - 1);
    
    // 2. Reverse each individual word
    reverseEachWord(charArray, n);
    
    // 3. Clean up spaces (Implementation omitted for brevity)
    return cleanSpaces(charArray, n);
}

private void reverse(char[] a, int i, int j) {
    while (i < j) {
        char t = a[i]; a[i++] = a[j]; a[j--] = t;
    }
}

Efficiency

Time: O(n). Linear time to reverse and clean.

Space: O(n). (Java strings are immutable, so we must use O(n) space anyway).

The beauty of this approach is that if strings were mutable, this would be an O(1) space solution. It's a classic algorithmic pattern.

⚡ Cheat Sheet

  • • Pattern: Reverse Scan (collect words from back to front).
  • • Regex: "\\s+" matches one or more whitespace characters.
  • • Space Trick: If allowed to use O(1) space, always mention the "Double Reversal" technique.
  • • Edge Case: All spaces string or leading/trailing spaces.

🧩 Edge Cases

  • " a " → Single letter surrounded by spaces.
  • " " → Empty word array (Handle split cautiously).
  • "a b" → Multiple internal spaces (Tricky pointers).
